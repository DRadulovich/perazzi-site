Perazzi Home Page – Component Summary (Top → Bottom)
ACT I — Hero (“Invitation”)
Component: HeroBanner
Purpose: Full-viewport hero introducing the brand’s essence and emotional tone.
Content: Tagline or mantra, optional subheading, background image or looping video, centered layout.
Behavior: Fades in on load, static until scroll. Background supports parallax drift (disabled for reduced-motion users).
Performance: Preloaded and optimized as the page’s LCP element.
Sub-component: ScrollIndicator
Purpose: Small UI cue (typically an SVG arrow or “scroll ↓” text) encouraging the user to continue downward.
Behavior: Decorative, slight bounce animation.
Accessibility: Hidden from screen readers if decorative; keyboard-focusable if interactive.

ACT II — Craftsmanship Journey (“Immersion in Craft”)
Component: TimelineScroller
Purpose: Manages the scroll-driven narrative that showcases Perazzi’s bespoke build process.
Behavior: On desktop, uses pinned scrolling; on mobile, linear stacked layout.
Function: Registers scroll depth analytics for each stage.
Contains: Multiple TimelineItem sub-components.
Sub-component: TimelineItem
Purpose: Represents one stage in the craftsmanship timeline.
Content: Title, descriptive text, associated image or video from CMS (“FactoryAsset”).
Behavior: Fades and slides into view on scroll (using Intersection Observer + Framer Motion).
Structure: Repeats per stage (e.g. “Measurement & Fit,” “Engraving the Soul,” etc.).
Accessibility: All content remains in DOM order; degrades gracefully if JS or animations are disabled.

ACT III — Champion Legacy (“Belonging”)
Component: MarqueeFeature
Purpose: Highlights a single champion or legacy story that embodies the brand’s heritage.
Content: Portrait or action photo, short quote, name/title, optional “Read Full Story” link.
Behavior: Static section with light cinematic reveal (fade/slide); hover or tap animates quote subtly.
Data Source: Champion document in CMS, optionally linked to Article.
Accessibility: Quote marked up with <blockquote> and <cite> if name shown; text never embedded in image.

ACT IV — Finale CTA (“Join the Legacy”)
Component: CTASection
Purpose: Closing section with a final brand statement and a primary call-to-action button (e.g. “Begin Your Journey”).
Layout: Centered text and button, subtle textured or dark background.
Behavior: Simple fade-up reveal as user scrolls in; button animates slightly on hover/focus.
Performance: Lightweight; link is prefetched for instant navigation.
Accessibility: Proper ARIA roles and focus styles; high-contrast text and button states.

GLOBAL CONTEXT
Header/Footer: Standard site components not included in this page spec but always present globally.
Framework: Next.js (App Router, React 18).
Styling: Tailwind CSS + Radix UI primitives.
Animation: Framer Motion for orchestrated sequences; CSS transitions for light effects.
CMS Bindings: Sanity CMS providing FittingStage, Champion, FactoryAsset, and Article data.
Hosting: Vercel deployment with Cloudinary for media delivery.
Accessibility and Performance: All sections meet WCAG contrast, keyboard navigation, and reduced-motion compliance; target LCP ≤ 2.5 s, CLS ≤ 0.05.
