# Shotguns – Component Prop Contracts (Condensed)

> These interfaces summarize expected props. Keep names/types consistent with `src/types/{content, catalog}.ts`. All media use `FactoryAsset` and must reserve `aspectRatio` to avoid CLS.

## Common Types
- `FactoryAsset { id: string; kind: "image"|"video"; url: string; alt: string; caption?: string; aspectRatio?: number }`
